CSV-Assistent

With this app, you can quickly and easily manage your CSV files and prepare them for export. You load one or more CSV files, and they are clearly displayed in a table. This allows you to view the contents directly and search for specific entries or terms. For exporting, you first create a custom target list – in other words, the columns that your new CSV file should have. For each loaded file, you can then map the columns of the original file to the columns of your target list. This way, you can precisely define which data goes where, and then export everything as a new, consistent CSV file. If all files already have the same format (the same columns), it’s even easier: in that case, you can export them directly without any additional mapping. The app helps you load, search, and standardize multiple CSV files and conveniently export them as a new, uniform CSV file.
