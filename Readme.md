CSV-Assistent
==============

Der CSV-Assistent ist ein Desktop-Tool, mit dem du mehrere CSV-Dateien schnell laden, durchsehen und für den Export vereinheitlichen kannst. Es richtet sich an alle, die regelmäßig Datenlisten zusammenführen oder in ein einheitliches Format bringen müssen.

Was du damit machen kannst
-------------------------
- **Mehrere CSVs gleichzeitig laden:** Öffne eine oder mehrere Dateien und betrachte deren Inhalt sofort in einer übersichtlichen Tabelle.
- **Einträge durchsuchen:** Finde gezielt Werte oder Begriffe, ohne die Datei extern zu bearbeiten.
- **Zielspalten festlegen:** Lege eine individuelle Zielliste an, die die Spalten deines finalen Exports beschreibt.
- **Spalten zuordnen:** Ordne für jede geladene Datei die vorhandenen Spalten deiner Zielliste zu, damit alle Daten ins gleiche Format überführt werden.
- **Direkt exportieren:** Falls deine Dateien bereits identische Spalten haben, exportierst du sie ohne zusätzlichen Aufwand als neue, einheitliche CSV.
- **Format vereinheitlichen:** Standardisiere unterschiedlich strukturierte CSVs und speichere sie gebündelt in einem sauberen Ausgabeformat.

Kurzer Ablauf
-------------
1. CSV-Dateien laden und Inhalte prüfen.
2. Zielspalten definieren, die deine Ausgabedatei enthalten soll.
3. Spalten jeder Quelle auf die Zielspalten mappen.
4. Alles als neue, konsistente CSV exportieren.

So hilft dir das Tool
---------------------
- Spart Zeit beim Zusammenführen verschiedener Datenquellen.
- Verhindert Fehler durch klare Zuordnung von Spalten.
- Produziert verlässliche, einheitliche CSV-Dateien für nachgelagerte Systeme oder Reports.

Dieses Tool enthält keine Werbung, kein Tracking und keine versteckten Monetarisierungsmechanismen.
